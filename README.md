## STDS Tasks Overview

Each task folder within the `STDS` directory contains specific deliverables, including the task description (`TaskDescription.md`), student solution files, and a reflection on the learning experience (`Reflection.md`).

### Task01_Excel_Analysis

This folder contains files related to Excel-based data analysis tasks.

*   `HR Data (1).xlsx`: Human Resources data for analysis.
*   `تحليل خاص بالمواد الغذائية.xlsx`: Foodstuffs analysis spreadsheet.
*   `TaskDescription.md`: Details about the Excel analysis task.
*   `Reflection.md`: Student's reflections on Task 01.

### Task02_AdventureWorks_PBI

This folder is dedicated to tasks involving Power BI analysis with the Adventure Works dataset.

*   `Adventure Works dash (1).pbix`: Power BI report for Adventure Works.
*   `Product (1).csv`: Product data for Adventure Works.
*   `Employee (1).csv`: Employee data for Adventure Works.
*   `SalesTerritory (1).csv`: Sales territory data for Adventure Works.
*   `ShipMethod (1).csv`: Shipping method data for Adventure Works.
*   `SalesView.csv`: Sales view data for Adventure Works.
*   `Sql_Task (1).sql`: SQL script related to Adventure Works data.
*   `TaskDescription.md`: Details about the Adventure Works Power BI task.
*   `Reflection.md`: Student's reflections on Task 02.

### Task03_SalesMarketing_PBI

This section covers Power BI tasks focused on Sales and Marketing data.

*   `Sales Marketing Dashboard.pbix`: Power BI dashboard for Sales and Marketing.
*   `executive_summary.png`: Executive summary image.
*   `customer_analysis.png`: Customer analysis image.
*   `marketing_conversion.png`: Marketing conversion image.
*   `sales_product_analysis.png`: Sales product analysis image.
*   `TaskDescription.md`: Details about the Sales & Marketing Power BI task.
*   `Reflection.md`: Student's reflections on Task 03.

### Task04_Tableau_Dashboard

This folder contains files for Tableau dashboard creation tasks.

*   `Dashboard (1).twb`: Tableau workbook file.
*   `map_json_countries.json`: GeoJSON file for country mapping.
*   `TaskDescription.md`: Details about the Tableau Dashboard task.
*   `Reflection.md`: Student's reflections on Task 04.
